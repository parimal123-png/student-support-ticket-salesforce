# Student Support Ticket Management System using Salesforce

## Project Description
This project is built using Salesforce Lightning Platform.
It allows students to raise support tickets, track their status,
and enables admins to manage and resolve issues efficiently.

## Features
- Custom Object: Support Ticket
- Fields: Issue Type, Priority, Status, Description
- Automation using Salesforce Flow
- Reports for ticket tracking

## Technology Used
- Salesforce Lightning
- No-Code Automation (Flows)
- Reports & Dashboards

## Note
This is a Salesforce no-code project.
Implementation is done inside Salesforce Developer Org,
hence no source code is available in this repository.
